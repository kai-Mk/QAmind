実装内容をレビューし、完了条件を確認するコマンド。

## 実行手順

1. 現在のブランチ名からステアリングファイルを特定する（ブランチ `feature/3-xxx` → `docs/steering/3-xxx.md`）
2. ステアリングファイルの完了条件を読み取る
3. サブエージェントでテストを実行する:
   - `pnpm test -- --run` - テスト実行
4. ステアリングファイルの完了条件と実装内容を照合する
5. `docs/TODO.md` の該当タスクのステータスを更新する
6. 結果を報告する:
   - 全て問題なし → 「pushしてPRをマージできます」と報告
   - 問題あり → 問題点を具体的に指摘する

## 注意事項

- テスト実行やビルド確認は必ずTaskツールでサブエージェントに委譲すること
- 完了条件を満たした項目はステアリングファイルのチェックボックスを `[x]` に自動更新する
- 全て問題なしの場合はコミット・pushまで自動で行う。マージはユーザーの判断に委ねる
